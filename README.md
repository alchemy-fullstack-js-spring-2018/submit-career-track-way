Project description: 
Calculator
===
In case you forgot how to do math, this library will help you out.

## Install

If/when published to npm, will let you know. 

```sh
> npm i fancy-calc
```

## API

### `calc.add(x,y)`
Returns the sume of two numbers:

```js
const calc = require ('calculator');// if it WAS on npm
calc.add(1,2);
// 3
```
